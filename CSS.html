Cascading Style Sheets

CSS stands for Cascading Style Sheets.

Cascading allows to apply style on page.

Style sheets make web page beautiful.

Why We Use Css

By css we can apply specific style to specific element.
In css all formatting and styles are placed in one place.
While in html all formatting and styles are stored in a single file which create difficult when our code is too long.

Inline Css

Inline css is used to insert style only a single element to make them unique.

In inline css style attribute is used to style element.

Embedded Css/Internal Css

Internal css is used to style one unique page.

In internal css style element is used.
In internal css if we paragraph is red than all para element will we red.

External CSS

In external css all formatting and styles are stored in a single file extension .css.

CSS Syntax

Css apply and browser interpets and applied to correspodding elements. Css syntax has three parts selector,property and value.

Id Selectors

Id selector is used to select element by using id.In style id start by hastag sign.

Class selectors

Class selectors is similar to id selectors.

Do not start id or class name by number.

Descendant Selectors

Descendant selectors is used to select that are descendant of another element.

Css Comments

Css is used to explain your code and help later when you open file.Comments are ingorned by browser.

Inheritance

Inherit means child element had usually have some property of parent element.Oterwise you define child element.

Font Family Property

Font family property is used to specific font of text.

There are two types of font family-

1-Font family in which font is different.

2-Generic font family in font is similar.

Font Size Property

Font size property is used to set font in web page.Font size can be change by keyboards or pixel.

Font Style Property

Font style property is used to set font style in page.
There are three values is italic,normal and oblique.

Oblique is similar to italic but less supported in browser.

Font Weight Property

Font weight property controls the boldness and thickness of text.There are three values by keyboard is bold,bolder,lighter.In number we can use 100 to 900.

Font Variant Property

Font variant allows to set font all small caps.There are three values normal,inherit,small caps.

Color Property

Color property specific color of font.Color name,hexadecimal,rgb are supported by browser.

Text Align Property

Text align property specific horizontal text in page.There are three values left,right,center and justify.

Vertical Align Property

Vertical align property is used to align vertical text ib web page.We can use values like top,middle,botton.

Also we can use like baseline,sub,super,pixel.

Text Decoration Property

Text decoration is specifies text decoration in page.Value are none,inherit,overline, underline,line-through.

Text Indent Property

Text Indent Property specifies indent of text of first line.

Text Shadow Property

Text shadow property is used to set shadow on text.First value is horizontal distance,second is vertical distance,third is blur value and fourth is color value.Negative value is supported.

Text Transform Property

Text Transform Property is used to transform into capitalize,lowercase and uppercase.

Letter Spacing Property

Letter spacing property is used to make space between letter.Negative values also supported.Values are inherit,normal and length.

Word Spacing Property

Word spacing property is used to make space between words in web page.Negative values also supported.Values are inherit,normal and length.

White Spacing Property

White space property is used to make space.Value are inherit,normal and nowrap.Also pre,pre-line and pre-wrap.

Introduction CSS Box Model

All elements in html are considered as box.Box contains content after that padding after that border after that margin.The design follow top,right,bottom,left.

The image shows all properties.

More On Box Model

All elements are box in html.Height and weight we write is content height and weight but when we set background color it cover content as well as padding.

Borders

Border property is used to draw border in web page.By default border is none.There are various types of design are given like none,solid,grove,dotted, dashed,double,ridge,inset and ouset.

Height And Weight Property

Height and weight apply on box according their requirement.

Background Color Property

Background color is specify background color to element.

Background Image Property

Background image specify to apply image in background or elment.

Background Repeat Property

Background repeat property specify to repeat image horizontal and vertical or both axes.

By default repeat both axes.Values is no-repeat,repeat-x and repeat-y.

Background Attachment Property

Background attachment property specify background attachment.Value is inherit,fixed and scroll.

List Style Type Property

List style type property is used to describe list type in web page.

List Style Position Property

List style position property is used to describe position of web page.Inside and outside two values.

List Style Image Property

List style Image property is used to describe image in item marker in web page.

Table Border Collapse Property

Table border collapse is used to make space in border or a line border.By default border collapse is separate.

Caption Side Property

Caption side is used to describe caption

Styling The Link

Link is used to connect web page.Value a:link,a:active,a:visited, a:hover.
